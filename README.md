# 🎮 Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game built with Python.

## 📌 Features

- Play against the computer
- Random computer choices using Python's `random` module
- ASCII art for Rock, Paper, and Scissors
- Win, lose, and tie detection
- Input validation

## 🛠️ Technologies Used

- Python 3
- random module

## 📂 Project Structure

```
Rock-Paper-Scissors-Game/
│── main.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/shubhamss006/Rock-Paper-Scissors-Game.git
```

2. Open the project folder.

3. Run:

```bash
python main.py
```
